# HribSpot
Quickly save mushroom species, quantity, and location while foraging offline

App made with tasker: <a href="https://taskernet.com/shares/?user=AS35m8lAGpgBQ83OXLWZDvG6p8apkrN%2F1fP55SfSMPnL9nfjqOB6XzKERt34IoSd%2FTwoHVexw1ep&id=Project%3AHribSpot">HribSpot taskernet.com</a>

1. Select mushroom species

2. Select the quantity found

3. Wait until location is acquired

4. If the previously saved mushroom needs to be edited (e.g. if you found more mushrooms in same ~20ft area where you already saved the location and want to update the quantity) hold on the "Select Mushroom" title to enter into editing mode OR just save another waypoint

5. When your finished foraging, click on the file icon and click in the year/month folder and open the GPX file with GaiaGPS
(waypoint colors must be updated manually in GaiaGPS as it is currently impossible to import gpx files with color)
Tip: Use the GaiaGPS website to quickly and efficiently change icon colors
